# Tarea 3 Inteligencia Artificial

## Si no existe crear entorno virtual

```
python3 -m venv .venv
```

## Desplegar entorno virtual

```
source .venv/bin/activate
```

## Desplegar dependencias para mac

```
pip install -r requirements.txt
```

## Desplegar dependencias dependiendo de su sistema operativo

Para Windows
```
pip install -r requirementsWindows.txt
```

Para Linux
```
pip install -r requirements_linux_rtx3060.txt
```

Para Mac
```
pip install -r requirementsMac.txt
```
